We are building a warehouse of ALL skills as more as we can find on the Internet. You can finish this with following steps:
- Search for agentic skills repo or collections, such as https://www.skillhub.club/skills.
- Crawl and download all the skills content locally
- Saven skills into <cwd>/downloaded_skills/<source_name>/<skills>. Such as for SkillHub, we download all skills into `<cwd>/downloaded_skills/skillhub/skill-a`, `<cwd>/downloaded_skills/skillhub/skill-b` and more
